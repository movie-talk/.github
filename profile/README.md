## 주요 기능

- **로그인 및 회원가입**: 세션 기반 로그인, 로그아웃, 회원가입 구현
- **보안**: 비밀번호 보안을 위해 Salt 및 Hash 기법 적용, 안전한 사용자 인증 시스템 구현
- **리뷰 CRUD**: 리뷰 등록, 수정, 삭제 조회 구현, MySQL을 활용한 데이터 저장 및 MyBatis를 활용한 데이터 매핑
